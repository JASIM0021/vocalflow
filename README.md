# VocalFlow

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Status](https://img.shields.io/badge/status-Open--Source-green.svg)

**VocalFlow** is a sleek, AI-powered Text-to-Speech (TTS) application designed to transform text into natural, lifelike speech instantly. Leveraging the **Azure Neural TTS engine**, it provides professional-grade voiceovers without the need for accounts or subscriptions.

---

## 🚀 Features

- **Neural AI Voices:** Access over **300+ high-quality voices** that sound human, not robotic.
- **Global Support:** Speak to the world with support for **50+ languages**.
- **Zero Barriers:** No sign-up required. Completely free and open-source.
- **Instant Export:** Download your generated audio as an **MP3** file with one click.
- **Dark Mode Support:** Toggle between light and dark themes for a comfortable workspace.
- **Keyboard Driven:** Optimized for speed with built-in shortcuts.

---

## ⌨️ Keyboard Shortcuts

Streamline your workflow with these "Pro Tip" shortcuts:

| Action              | Shortcut         |
| :------------------ | :--------------- |
| **Generate Speech** | `Ctrl` + `Enter` |
| **Play / Pause**    | `Space`          |

---

## 🛠️ How it Works

1.  **Input Text:** Enter up to 5,000 characters in the text area.
2.  **Select Voice:** Choose your preferred voice (e.g., _Aria_) and language.
3.  **Generate:** Hit the **Generate Speech** button or use the shortcut.
4.  **Download:** Listen to your audio and export it as an MP3 instantly.
